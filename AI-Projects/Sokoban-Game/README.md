# Sokoban AI Solver 📦🤖

An advanced Python implementation of the Sokoban puzzle game that compares different Artificial Intelligence search strategies to find the optimal solution.

## 🚀 AI Algorithms Implemented
This project includes three core search algorithms to solve the puzzles:
1. **BFS (Breadth-First Search):** Guarantees the shortest path by exploring all nodes at the current depth before moving deeper.
2. **DFS (Depth-First Search):** Explores as far as possible along each branch before backtracking (useful for memory efficiency in some cases).
3. **A* (A-Star Search):** The most efficient solver using **Heuristics** to find the optimal path quickly by estimating the cost to the goal.

## 🛠️ Technical Stack
* **Language:** Python 🐍
* **Concepts:** State Space Search, Heuristic Functions, Pathfinding.
* **Environment:** Terminal / GUI based implementation.
